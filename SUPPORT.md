# Support

## Getting Help

For setup, configuration, or usage help with this project:

- review `README.md` first
- check `SECURITY.md` for vulnerability reporting guidance
- review `CONTRIBUTING.md` before proposing code changes

## Recommended Support Request Format

When asking for help, include:

- Windows version
- Python version
- the command you ran
- the error message or log snippet
- whether the issue is with monitoring, detection, alerting, or runtime control

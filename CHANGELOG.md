# Changelog

All notable changes to this project will be documented in this file.

## [1.0.0] - 2026-03-25

### Added

- initial clean-architecture backend implementation
- Windows Security Event Log reader for RDP-related `4624` and `4625` events
- normalized event parser and typed models
- brute-force, suspicious IP, and rate anomaly detection rules
- Discord webhook alerting with retry and throttling
- structured rotating JSON logs
- YAML and `.env` configuration support
- CLI service controls for `start`, `stop`, and `status`
- repository documentation and BSM project branding

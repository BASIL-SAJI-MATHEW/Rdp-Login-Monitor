# Code of Conduct

This repository aims to be professional, respectful, and constructive.

## Expected Behavior

- communicate with respect
- give technical feedback clearly and professionally
- assume good intent while staying honest about risks
- keep discussions focused on improving the project

## Unacceptable Behavior

- harassment or discriminatory behavior
- abuse, threats, or personal attacks
- intentional sabotage of collaboration
- plagiarism or removal of authorship notices
